# 📊 Smart Data Analyzer

An AI-powered web application that enables users to upload CSV datasets and perform automated data analysis, cleaning, visualization, and AI-driven insights without writing any code.

## ✨ Features

- 📁 Upload and analyze any CSV dataset
- 🔍 Automated data profiling
- 🧹 One-click data cleaning
- 📈 Interactive data visualizations (7+ chart types)
- 🤖 AI-generated business insights using Llama 3.1
- 💬 Ask natural language questions about your data
- 📥 Download the cleaned dataset

## 🛠️ Tech Stack

- **Language:** Python 3.12
- **Frontend:** Streamlit, HTML, CSS
- **Data Processing:** Pandas, NumPy
- **Visualization:** Plotly Express
- **AI:** Groq API, Llama 3.1 (8B)
- **Configuration:** python-dotenv

````

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/gaurikad11/Smart-Data-Analyzer.git
cd Smart-Data-Analyzer
````

2. Create a virtual environment:

```bash
python -m venv venv
```

3. Activate the environment:

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

4. Install dependencies:

```bash
pip install -r requirements.txt
```

5. Create a `.env` file and add your Groq API key:

```env
GROQ_API_KEY=your_groq_api_key_here
```

6. Run the application:

```bash
streamlit run app.py
```
